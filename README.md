# SECURE-DEVOPS-PRACTICES

##  Description
This project demonstrates Secure DevOps practices using CI/CD pipeline integrated with security scanning tools.


## Tools Used
- GitHub Actions (CI/CD)
- SonarQube (Static Code Analysis)
- OWASP ZAP (Security Testing)



## Pipeline Flow
1. Code push triggers pipeline
2. SonarQube scans code quality
3. OWASP ZAP checks vulnerabilities
4. Reports generated automatically



## Output Files
- vulnerability-report.md
- zap-baseline-report.html



##  Result
Security scanning successfully integrated into CI/CD pipeline.
